# Life Infinity Loop Runner

Read life-infinity-loop-profile.md

1. Identify next_domain
2. Ask ONE diagnostic question
3. Wait for user response
4. Evaluate:
   - score
   - gap
   - mechanism
5. Update profile:
   - last_question
   - last_score
   - gaps
   - mechanisms_taught
6. Append to logs/YYYY-MM.md
7. Rotate domain

## Rotation

Love → Work → Play → Health → Spiritual

If score ≤ 5:
repeat domain once before rotating

## Weighting

For first 6–12 months, bias questions toward:
- Play (under-owned historically)
- Spiritual (growth area)

Love, Work, Health have 10 years of muscle — trust the foundation, but don't neglect.
