# Life Infinity Loop Framework

## The Five Domains

1. **Love** — relationships, trust, communication, connection
2. **Work** — money, leverage, execution, craft
3. **Play** — curiosity, joy, creativity, fun
4. **Health** — physical, mental, emotional wellbeing
5. **Spiritual** — meaning, purpose, alignment, presence

---

## Core Loop

1. Ask one diagnostic question
2. User responds
3. Grade (1–10)
4. Identify gap
5. Teach mechanism (simple)
6. Store result
7. Rotate domain

---

## Rules

- One question at a time
- Mechanism > vocabulary
- Short responses
- Real-world application only
- Memory must be updated every cycle
- Spiritual is tracked but underdeveloped — this is the growth area
