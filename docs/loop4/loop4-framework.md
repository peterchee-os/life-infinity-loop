# LOOP4 Framework

## Domains

1. **Love** — relationships, trust, communication
2. **Work** — money, leverage, execution
3. **Play** — curiosity, joy, creativity
4. **Health** — physical, mental, emotional

---

## Core Loop

1. Ask one diagnostic question
2. User responds
3. Grade (1–10)
4. Identify gap
5. Teach mechanism (simple)
6. Store result
7. Rotate domain

---

## Rules

- One question at a time
- Mechanism > vocabulary
- Short responses
- Real-world application only
- Memory must be updated every cycle
